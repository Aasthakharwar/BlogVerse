
# 📝 BlogVerse with AI Integration

**BlogVerse** is a blogging platform enhanced with **Google Gemini AI** for content generation.  
It supports two roles: **Admin** and **User**, each with distinct features.

---

## ✨ Features

### 🔑 Admin
- Create blogs manually or with AI assistance  
- Publish, unpublish, or delete blogs  
- Moderate user comments  

### 👥 User
- Read published blogs  
- Submit comments (subject to admin approval)  

---

## ⚙️ Tech Stack
- React.js + Tailwind CSS  
- Node.js + Express.js  
- MongoDB  
- Google Gemini API
