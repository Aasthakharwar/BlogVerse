# 📝 BlogVerse – AI-Powered Blogging Platform

BlogVerse is a full-stack blogging platform that enables users to explore, read, and interact with blogs while providing administrators with advanced content management capabilities. The platform integrates **Google Gemini AI** to assist in generating blog content, making content creation faster and more efficient.

---

## ✨ Features

### 🔑 Admin
- Create blogs manually or generate content using Google Gemini AI
- Edit, publish, unpublish, and delete blog posts
- Moderate and manage user comments
- Secure admin dashboard for content management

### 👥 User
- Browse and read published blogs
- View detailed blog posts
- Submit comments on blogs
- Responsive and user-friendly interface

---

## 🚀 Key Highlights

- AI-assisted blog generation using Google Gemini API
- Role-based authentication for Admin and User
- Secure RESTful API integration
- Comment moderation system

- Clean and intuitive user interface

## 🔒 Environment Variables

Create a `.env` file in the server directory and configure the required environment variables.

Example:

```env
MONGODB_URI=
JWT_SECRET=
GEMINI_API_KEY=
```
